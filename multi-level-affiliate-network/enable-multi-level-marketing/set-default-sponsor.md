# Set Default Sponsor

**GoAffPro** provides you with the option to set the default sponsor for the multi level network.&#x20;

To set the default sponsor, go to the **Multi Level** tab in the GoAffPro admin panel.&#x20;

<figure><img src="../../.gitbook/assets/image (3831).png" alt=""><figcaption><p>Multi Level</p></figcaption></figure>

Here, go to the Default Sponsor section.&#x20;

<figure><img src="../../.gitbook/assets/image (3832).png" alt=""><figcaption><p>Default Sponsor</p></figcaption></figure>

Now, select the affiliate.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-01-14 1907244.png" alt=""><figcaption><p>Select the affiliate</p></figcaption></figure>

After this, any new affiliate who is not sponsored will be placed under this affiliate as a downline.&#x20;
