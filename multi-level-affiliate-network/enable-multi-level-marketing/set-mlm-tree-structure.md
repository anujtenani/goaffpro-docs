# Set MLM Tree Structure

**GoAffPro** provides you with the option to set the tree structure for your MLM network.&#x20;

To set the tree structure for your MLM network, go to the **Multi Level** tab in the GoAffPro admin panel.&#x20;

<figure><img src="../../.gitbook/assets/image (3831).png" alt=""><figcaption><p>Multi Level</p></figcaption></figure>

Here, go to the Recruitment Mode section.

<figure><img src="../../.gitbook/assets/image (3834).png" alt=""><figcaption><p>Recruitment Mode</p></figcaption></figure>

Now, select the tree structure.&#x20;

{% hint style="info" %}
You can select between Unilevel (default) or Forced matrix structures.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2026-01-28 214826.png" alt=""><figcaption><p>Select the tree structure</p></figcaption></figure>

After this, set the width and depth of the network.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-01-28 214831.png" alt=""><figcaption><p>Set the width and depth of the network</p></figcaption></figure>

Finally, click on **Submit**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-01-28 2148352.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The tree structure with this depth and width will be set for the MLM network.

<figure><img src="../../.gitbook/assets/image (3835).png" alt=""><figcaption></figcaption></figure>

<details>

<summary>FAQ</summary>

**What is the default recruitment mode and structure?**

The default configuration is the Unilevel structure.

**How to set a binary configuration for the MLM network?**

To configure a binary setup, set the width to two and leave the depth option blank.

</details>
