# Setup Recurring Bonus for Affiliates

**GoAffPro** provides you with the option to set up a recurring bonus for affiliates.&#x20;

{% hint style="info" %}
The recurring bonus feature lets you set up a bonus for affiliates each time the target is reached. It ensures that your affiliates stay motivated to promote your products or services consistently.
{% endhint %}

To set up a recurring bonus for affiliates, go to the **Recurring Bonus** section in the **Targets** tab of the GoAffPro admin panel.

<figure><img src="../.gitbook/assets/image (120) (1).png" alt=""><figcaption><p>Targets > Recurring bonus</p></figcaption></figure>

Here, click on **Create new recurring bonus**.

<figure><img src="../.gitbook/assets/Screenshot 2026-03-03 212246.png" alt=""><figcaption><p>Click on Create new recurring bonus</p></figcaption></figure>

This will open up the Create recurring bonus window.

<figure><img src="../.gitbook/assets/image (3840).png" alt=""><figcaption><p>Create recurring bonus</p></figcaption></figure>

Here, set the bonus title.&#x20;

<figure><img src="../.gitbook/assets/Screenshot 2026-03-03 212251.png" alt=""><figcaption><p>Set the bonus title</p></figcaption></figure>

Now, set when to give the bonus.

{% hint style="info" %}
You have the option to give the bonus daily, weekly, monthly, or quarterly.
{% endhint %}

<figure><img src="../.gitbook/assets/Screenshot 2026-03-03 212255.png" alt=""><figcaption><p>Set when to give the bonus</p></figcaption></figure>

Next, set the date and time for the first bonus to be given.

<figure><img src="../.gitbook/assets/Screenshot 2026-03-03 2137522.png" alt=""><figcaption><p>Set the date and time for the first bonus to be given</p></figcaption></figure>

You can also set the bonus data end date.&#x20;

<figure><img src="../.gitbook/assets/Screenshot 2026-03-03 212301.png" alt=""><figcaption><p>Set the bonus data end date</p></figcaption></figure>

After this, select the parameters for the bonus.&#x20;

{% hint style="info" %}
You can select parameters such as Number of orders, Total sale value (subtotal), Total sale value (total), or Total Commission.
{% endhint %}

<figure><img src="../.gitbook/assets/Screenshot 2026-03-03 212431.png" alt=""><figcaption><p>Select parameters for the bonus</p></figcaption></figure>

Next, set the bonus amount.&#x20;

{% hint style="info" %}
You can choose to set the bonus as either a fixed amount or a percentage.
{% endhint %}

<figure><img src="../.gitbook/assets/Screenshot 2026-03-03 212436.png" alt=""><figcaption><p>Set the bonus amount</p></figcaption></figure>

You also have the option to choose how to give the bonus, either as a cash bonus or a discount coupon.

<figure><img src="../.gitbook/assets/Screenshot 2026-03-03 212443.png" alt=""><figcaption><p>Select how you want to give the bonus</p></figcaption></figure>

Finally, click on **Submit**.&#x20;

<figure><img src="../.gitbook/assets/Screenshot 2026-03-03 212455.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The recurring bonus will be set for affiliates in the program.

<figure><img src="../.gitbook/assets/image (3839).png" alt=""><figcaption></figcaption></figure>
