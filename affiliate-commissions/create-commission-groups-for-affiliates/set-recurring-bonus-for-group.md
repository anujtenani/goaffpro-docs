# Set Recurring Bonus for Group

**GoAffPro** provides you with the option to set up a recurring bonus for the commission group.&#x20;

To set up a recurring bonus for the commission group, go to the **Recurring Bonus** section in the **Targets** tab of the GoAffPro admin panel.&#x20;

<figure><img src="../../.gitbook/assets/image (3867).png" alt=""><figcaption><p>Targets > Recurring Bonus</p></figcaption></figure>

Here, click on **Create new recurring bonus**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-04-29 021756.png" alt=""><figcaption><p>Click on Create new recurring bonus</p></figcaption></figure>

This will open up the Create recurring bonus window.

<figure><img src="../../.gitbook/assets/image (3869).png" alt=""><figcaption><p>Create recurring bonus</p></figcaption></figure>

Here, set the bonus title.

<figure><img src="../../.gitbook/assets/Screenshot 2026-04-29 021800.png" alt=""><figcaption><p>Set the bonus title</p></figcaption></figure>

Now, set when to give the bonus.

{% hint style="info" %}
You have the option to give the bonus daily, weekly, monthly, or quarterly.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2026-04-29 021805.png" alt=""><figcaption><p>Set when to give the bonus</p></figcaption></figure>

Next, set the date and time for the first bonus to be given.

<figure><img src="../../.gitbook/assets/Screenshot 2026-04-29 021810.png" alt=""><figcaption><p>Set the date and time for the first bonus to be given</p></figcaption></figure>

You can also set the bonus data end date.

<figure><img src="../../.gitbook/assets/Screenshot 2026-04-29 021814.png" alt=""><figcaption><p>Set the bonus data end date</p></figcaption></figure>

Now, select the parameters for the bonus.

{% hint style="info" %}
You can select parameters such as Number of orders, Total sale value (subtotal), Total sale value (total), or Total Commission.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2026-04-29 021818.png" alt=""><figcaption><p>Select parameters for the bonus</p></figcaption></figure>

Next, set the bonus amount.

{% hint style="info" %}
You can choose to set the bonus as either a fixed amount or a percentage.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2026-04-29 021823.png" alt=""><figcaption><p>Set the bonus amount</p></figcaption></figure>

You also have the option to choose how to give the bonus, either as a cash bonus or a discount coupon.

<figure><img src="../../.gitbook/assets/Screenshot 2026-04-29 021827.png" alt=""><figcaption><p>Select how you want to give the bonus</p></figcaption></figure>

After this, select the commission group.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-04-29 021831.png" alt=""><figcaption><p>Select the commission group</p></figcaption></figure>

Finally, click on **Submit**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-04-29 0218354.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The recurring bonus will be set for the group.&#x20;

<figure><img src="../../.gitbook/assets/image (3868).png" alt=""><figcaption></figcaption></figure>
