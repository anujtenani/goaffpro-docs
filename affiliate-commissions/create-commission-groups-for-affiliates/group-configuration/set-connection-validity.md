# Set Connection Validity for Group

**GoAffPro** provides you with the option to set the customer-affiliate connection validity for the commission group.

To set the customer-affiliate connection validity for the commission group, go to the **Groups** tab in the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (177) (1).png" alt=""><figcaption><p>Groups</p></figcaption></figure>

Here, click on **Edit Group**.

![Click on Edit Group](<../../../.gitbook/assets/Screenshot 2021-08-17 005319.png>)

This will open up the Edit group section.

![Edit Group](<../../../.gitbook/assets/image (479).png>)

Here, go to the Group Configuration section.

<figure><img src="../../../.gitbook/assets/image (3682).png" alt=""><figcaption><p>Group Configuration</p></figcaption></figure>

Now, click on **Edit**.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-18 220154.png" alt=""><figcaption><p>Click on Edit</p></figcaption></figure>

Next, enable the "**Automatically connect customer with affiliate**" option.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-18 220217 (1).png" alt=""><figcaption><p>Enable the "<strong>Automatically connect customer with affiliate</strong>" option</p></figcaption></figure>

After this, enable the "Set connection expiration?" option.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-18 2202386.png" alt=""><figcaption><p>Enable the "Set connection expiration?" option</p></figcaption></figure>

Set the connection validity.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-18 220326.png" alt=""><figcaption><p>Set the connection validity</p></figcaption></figure>

Finally, click on **OK**.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-18 220350.png" alt=""><figcaption></figcaption></figure>

The connections will automatically expire after the validity period in the group.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-18 220401.png" alt=""><figcaption></figcaption></figure>
