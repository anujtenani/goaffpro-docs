# Select Dashboard Tabs for Group

**GoAffPro** provides you with the option to select the dashboard tabs for the commission group.

To select the dashboard tabs for the commission group, go to the **Groups** tab in the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (182) (1).png" alt=""><figcaption><p>Groups</p></figcaption></figure>

Here, click on **Edit Group**.

![Click on Edit Group](<../../../.gitbook/assets/Screenshot 2021-08-30 165221.png>)

This will open up the Edit group section.

![Edit Group](<../../../.gitbook/assets/image (2863).png>)

Now, go to the Group Configuration section.

<figure><img src="../../../.gitbook/assets/image (3692).png" alt=""><figcaption><p>Group Configuration</p></figcaption></figure>

Next, click on **Edit** in the Dashboard pages option.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-24 173213.png" alt=""><figcaption><p>Dashboard pages > Click on Edit</p></figcaption></figure>

Here, select the dashboard tabs.

{% hint style="info" %}
To create new dashboard tabs, please view this [guide](../../../affiliate-portal-customization/affiliate-dashboard/create-new-dashboard-pages/).&#x20;
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-24 173547.png" alt=""><figcaption><p>Select the dashboard tabs</p></figcaption></figure>

After this, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-24 173617.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The dashboard tab will be added to the group affiliate dashboard.&#x20;

<figure><img src="../../../.gitbook/assets/image (3691).png" alt=""><figcaption><p>The dashboard tab will be added</p></figcaption></figure>

Finally, the dashboard tab will be displayed to the group affiliates.

![](<../../../.gitbook/assets/Screenshot 2021-08-30 170011.png>)
