# Set Custom Media Categories for Group

**GoAffPro** provides you with the option to set custom media categories for the commission group.&#x20;

To set custom media categories for the commission group, go to the **Groups** tab in the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (180) (1).png" alt=""><figcaption><p>Groups</p></figcaption></figure>

Here, click on **Edit Group**.

![Click on Edit Group](<../../../.gitbook/assets/image (3284).png>)

This will open up the Edit group section.

<figure><img src="../../../.gitbook/assets/image (3547).png" alt=""><figcaption><p>Edit Group</p></figcaption></figure>

Now, go to the Group Configuration section.

<figure><img src="../../../.gitbook/assets/image (81).png" alt=""><figcaption><p>Group Configuration</p></figcaption></figure>

Next, click on **Edit**.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-10-21 105356 (1).png" alt=""><figcaption><p>Click on Edit</p></figcaption></figure>

Next, enable the "**Group has custom media categories**" option.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-10-21 113656.png" alt=""><figcaption><p>Enable the "group has custom digital aseet categories" option</p></figcaption></figure>

Here, select the digital asset categories.

{% hint style="info" %}
To create digital asset categories, you can follow this [guide](../../../program-management/upload-digital-assets-for-affiliates/move-digital-assets-to-folders.md).
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2025-10-21 113710.png" alt=""><figcaption><p>Select the media assets</p></figcaption></figure>

After this, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-10-21 1137234.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

Finally, the media assets will be set for the group.

<figure><img src="../../../.gitbook/assets/image (80).png" alt=""><figcaption></figcaption></figure>
