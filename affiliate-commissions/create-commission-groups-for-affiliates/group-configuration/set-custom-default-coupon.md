# Set Custom Default Coupon for Group

**GoAffPro** provides you with the option to set a custom default coupon for the commission group.

To set a custom default coupon for the commission group,  go to the **Groups** tab in the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (179) (1).png" alt=""><figcaption><p>Groups</p></figcaption></figure>

Here, click on **Edit Group**.

<figure><img src="../../../.gitbook/assets/Screenshot 2023-06-26 144345.png" alt=""><figcaption><p>Click on Edit Group</p></figcaption></figure>

This will open up the Edit group section.

<figure><img src="../../../.gitbook/assets/image (2160).png" alt=""><figcaption><p>Edit Group</p></figcaption></figure>

Now, go to the Group Configuration section.

<figure><img src="../../../.gitbook/assets/image (196).png" alt=""><figcaption><p>Group Configuration</p></figcaption></figure>

Here, click on **Edit**.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-17 124330 (1).png" alt=""><figcaption><p>Click on Edit</p></figcaption></figure>

Next, select the custom default coupon option.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-17 124406 (1).png" alt=""><figcaption><p>Select the custom default coupon option</p></figcaption></figure>

After this, set the default discount code.

{% hint style="info" %}
You will need to create this discount code manually in the discount section of your store.
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-17 124423.png" alt=""><figcaption><p>Set the default discount code</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-17 124445.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The custom default coupon will be set for the commission group.

<figure><img src="../../../.gitbook/assets/image (198).png" alt=""><figcaption></figcaption></figure>
