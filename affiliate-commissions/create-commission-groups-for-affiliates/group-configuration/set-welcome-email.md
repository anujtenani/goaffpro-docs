# Set Welcome Email for Commission Group

**GoAffPro** provides you with the option to set up a welcome email for the commission group.

To set up a welcome email for the commission group, go to the **Signup page** section in the **Look and Feel** tab of the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (184) (1).png" alt=""><figcaption><p>Groups</p></figcaption></figure>

Here, click on **Edit Group**.

![Click on Edit Group](<../../../.gitbook/assets/Screenshot 2022-06-28 011549.png>)

This will open up the Edit group section.&#x20;

![Edit Group](<../../../.gitbook/assets/image (1634).png>)

Now, go to the Group Configuration section.

<figure><img src="../../../.gitbook/assets/image (220).png" alt=""><figcaption><p>Group Configuration</p></figcaption></figure>

Next, click on **Edit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-03 1441524.png" alt=""><figcaption><p>Click on Edit</p></figcaption></figure>

After this, select the welcome email.&#x20;

{% hint style="info" %}
You can create the welcome email for the group by following this [guide](../../../affiliate-email/set-email-notifications-for-affiliates/create-custom-email-notifications.md).
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-03 144240.png" alt=""><figcaption><p>Select the welcome email</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-03 145414.png" alt=""><figcaption></figcaption></figure>
