# Set Gifts for Group Affiliates

**GoAffPro** provides you with the option to set up gifts for the commission group.

To set up gifts for the commission group, go to the **Groups** tab in the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (3741).png" alt=""><figcaption><p>Groups</p></figcaption></figure>

Here, click on **Edit Group**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-05-12 224553.png" alt=""><figcaption><p>Click on Edit Group</p></figcaption></figure>

This will open up the Edit group section.

<figure><img src="../../../.gitbook/assets/image (3742).png" alt=""><figcaption><p>Edit Group</p></figcaption></figure>

Now, go to the Group Configuration section.

<figure><img src="../../../.gitbook/assets/image (3743).png" alt=""><figcaption><p>Group Configuration</p></figcaption></figure>

Next, click on **Edit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-05-12 224616.png" alt=""><figcaption><p>Click on Edit</p></figcaption></figure>

After this, select the gift template.

{% hint style="info" %}
You can follow this [guide](../../../affiliate-coupons/setup-gifts-for-affiliates/) to create the gift template.&#x20;
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2025-05-12 2246313.png" alt=""><figcaption><p>Select the gift template</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-05-12 2246444.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The gift template will be set for the group affiliates. They can redeem the gift products from their dashboard.&#x20;

<figure><img src="../../../.gitbook/assets/image (3740).png" alt=""><figcaption></figcaption></figure>
