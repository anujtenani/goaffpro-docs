# Set Group Badge

**GoAffPro** provides you with the option to set the group badge for the commission group.

To set the group badge for the commission group, go to the **Groups** tab in the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (181) (1).png" alt=""><figcaption><p>Groups</p></figcaption></figure>

Here, click on **Edit Group**.

![Click on Edit Group](<../../../.gitbook/assets/Screenshot 2021-06-07 142039.png>)

This will open up the Edit group section.

![Edit Group](<../../../.gitbook/assets/image (1692).png>)

Now, go to the Group Configuration section.

<figure><img src="../../../.gitbook/assets/image (3688).png" alt=""><figcaption><p>Group Configuration</p></figcaption></figure>

Next, click on the **Group Badge** option.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-24 140204.png" alt=""><figcaption><p>Click on the Group Badge option</p></figcaption></figure>

After this, upload the image file.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-24 140509.png" alt=""><figcaption><p>Upload the image file</p></figcaption></figure>

Finally, click on **OK**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-24 140542.png" alt=""><figcaption><p>Click on OK</p></figcaption></figure>

The group badge will be set for the group.&#x20;

<figure><img src="../../../.gitbook/assets/image (3689).png" alt=""><figcaption><p>Group badge will be set</p></figcaption></figure>

The badge will now appear in the affiliate dashboard.

<figure><img src="../../../.gitbook/assets/image (3690).png" alt=""><figcaption></figcaption></figure>
