# Set Payment Methods for Group

**GoAffPro** provides you with the option to set the payment methods for the commission group.

To set the payment methods for the commission group, go to the **Groups** tab in the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (178) (1).png" alt=""><figcaption><p>Groups</p></figcaption></figure>

Here, click on **Edit Group**.

![Click on Edit Group](<../../../.gitbook/assets/Screenshot 2021-06-07 142039.png>)

This will open up the Edit group section.

<figure><img src="../../../.gitbook/assets/image (3548).png" alt=""><figcaption><p>Edit Group</p></figcaption></figure>

Now, go to the Group Configuration section.

<figure><img src="../../../.gitbook/assets/image (3683).png" alt=""><figcaption><p>Group Configuration</p></figcaption></figure>

Next, click on **Edit**.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-19 004906.png" alt=""><figcaption><p>Click on Edit</p></figcaption></figure>

After this, select the payment methods.

{% hint style="info" %}
You can select multiple payment methods.
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-19 004926.png" alt=""><figcaption><p>Select the payment method</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-19 0049403.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

Finally, the payment methods will be set for the group.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-02-19 004952.png" alt=""><figcaption></figcaption></figure>
