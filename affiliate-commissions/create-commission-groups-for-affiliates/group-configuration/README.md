# Group Configuration

The **Group Configuration** section allows you to configure group settings for group affiliates. This includes options such as:

* [Set default referral link for the group](set-default-referral-link.md)
* [Set default network link for the group](set-default-network-link.md)
* [Set quick links for the group](set-quick-links.md)
* [Add instructions for the group affiliates](add-instructions-for-group-affiliates.md)
* [Set payment methods for the group](set-payment-methods.md)
* [Enable recurring commission for the group](enable-recurring-commission.md)
* [Set connection validity for the group](set-connection-validity.md)
* [Select dashboard tabs for the group](select-dashboard-tabs-for-group.md)
* [Set the group badge](set-group-badge.md)
* [Give commission only for new customers in the group](give-commission-only-for-new-customers.md)
* [Set welcome email for the commission group](set-welcome-email.md)
* [Set coupon template for the group](set-coupon-template.md)
* [Set custom default coupon for the group](set-custom-default-coupon.md)
* [Set gifts for group affiliates](set-gifts-for-group-affiliates.md)
* [Set custom media categories for the group](set-custom-media-categories-for-group.md)
* [Setup affiliate store for the group](setup-affiliate-store.md)

Go to the **Groups** > **Edit Group > Group Configuration** section.
