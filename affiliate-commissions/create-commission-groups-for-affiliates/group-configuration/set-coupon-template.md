# Set Coupon Template for Group

**GoAffPro** provides you with the option to set a coupon template for the commission group.

To set a coupon template for the commission group,  go to the **Groups** tab in the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (179) (1).png" alt=""><figcaption><p>Groups</p></figcaption></figure>

Here, click on **Edit Group**.

<figure><img src="../../../.gitbook/assets/Screenshot 2023-06-26 144345.png" alt=""><figcaption><p>Click on Edit Group</p></figcaption></figure>

This will open up the Edit group section.

<figure><img src="../../../.gitbook/assets/image (2160).png" alt=""><figcaption><p>Edit Group</p></figcaption></figure>

Now, go to the Group Configuration section.

<figure><img src="../../../.gitbook/assets/image (197).png" alt=""><figcaption><p>Group Configuration</p></figcaption></figure>

Here, click on **Edit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-17 124330.png" alt=""><figcaption><p>Click on Edit</p></figcaption></figure>

Next, select the coupon template option.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-17 124406.png" alt=""><figcaption><p>Select the coupon template option</p></figcaption></figure>

After this, select the coupon template.

{% hint style="info" %}
You can create a coupon template by following this [guide](../../../affiliate-coupons/setup-referral-coupons/create-coupon-templates.md).&#x20;
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-17 124902.png" alt=""><figcaption><p>Select the coupon template</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-03-17 1249513.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The coupon template will be set for the commission group.

<figure><img src="../../../.gitbook/assets/image (199).png" alt=""><figcaption></figcaption></figure>
