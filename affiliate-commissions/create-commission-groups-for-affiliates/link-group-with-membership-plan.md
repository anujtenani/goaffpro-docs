# Link Group with Membership Plan

**GoAffPro** provides you with the option to link the commission group with the membership plan.

To link the commission group with the membership plan, go to the **Membership Plans** tab in the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (3821).png" alt=""><figcaption><p>Membership Plans</p></figcaption></figure>

Here, click on **New payment plan**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-03 172633.png" alt=""><figcaption><p>Click on New payment plan</p></figcaption></figure>

This will open up the New plan section.

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-03 173117.png" alt=""><figcaption><p>New plan</p></figcaption></figure>

Now, set up the plan details.

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-03 173117 (1).png" alt=""><figcaption><p>Set up the plan details</p></figcaption></figure>

Next, select the commission group.

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-03 173245.png" alt=""><figcaption><p>Select the affiliate group</p></figcaption></figure>

After this, click on **Submit**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-03 173325.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

Affiliates who select this membership plan will automatically join the linked affiliate group.

<figure><img src="../../.gitbook/assets/image (3822).png" alt=""><figcaption></figcaption></figure>
