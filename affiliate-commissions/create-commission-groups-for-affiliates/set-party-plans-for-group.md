# Set Party Plans for Group

**GoAffPro** provides you with the option to set party plans for the commission group.&#x20;

To set party plans for the commission group, go to the **Groups** tab in the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (167) (1).png" alt=""><figcaption><p>Groups</p></figcaption></figure>

Here, click on **Edit Group**.

![Click on Edit Group](<../../.gitbook/assets/Screenshot 2022-06-13 185055.png>)

This will open up the edit group section.

<figure><img src="../../.gitbook/assets/image (3555).png" alt=""><figcaption><p>Edit Group</p></figcaption></figure>

Now, go to the Party links configuration section.

![Party links configuration](<../../.gitbook/assets/image (2547).png>)

Next, click on **Configure**.

![Click on Configure](<../../.gitbook/assets/Screenshot 2022-06-13 185404 (1).png>)

This will open up the Party plan configuration window.

<figure><img src="../../.gitbook/assets/image (3746).png" alt=""><figcaption><p>Party plan configuration</p></figcaption></figure>

Here, set the expiration for the party.&#x20;

{% hint style="info" %}
You can select either a fixed or a flexible expiration type.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2025-05-13 203047.png" alt=""><figcaption><p>Set the party expiration type</p></figcaption></figure>

Next, set the commission rate for affiliates on the party sales.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-05-13 203116.png" alt=""><figcaption><p>Set the commission rate for affiliates on party sales</p></figcaption></figure>

You can also set a host discount that they can offer to their party customers.

<figure><img src="../../.gitbook/assets/Screenshot 2025-05-13 203218.png" alt=""><figcaption><p>Set the host discount</p></figcaption></figure>

After this, set the host reward.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-05-13 203255.png" alt=""><figcaption><p>Set the reward</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-05-13 2032553.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The party plan will be set for the group affiliates.&#x20;

<figure><img src="../../.gitbook/assets/image (3747).png" alt=""><figcaption></figcaption></figure>
