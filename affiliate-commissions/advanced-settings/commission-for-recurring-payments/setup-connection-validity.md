# Setup Connection Validity

**GoAffPro** provides you with the option to set up the customer-affiliate connection validity.

To set up the customer-affiliate connection validity, go to the **Extras** section in the **Settings** tab of the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (3875).png" alt=""><figcaption><p>Settings > Extras</p></figcaption></figure>

Here, go to the Customer-Affiliate Connect section.

<figure><img src="../../../.gitbook/assets/image (3876).png" alt=""><figcaption><p>Customer-Affiliate Connect</p></figcaption></figure>

Now, enable the "**Set connection expiration (in days)?**" option.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2026-05-18 235530.png" alt=""><figcaption><p>Enable the "Set connection expiration (in days)?" option</p></figcaption></figure>

Next, set the connection validity.

{% hint style="info" %}
Leaving the "Connection expires after" field empty will set the connection validity to lifetime.
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2026-05-18 235537.png" alt=""><figcaption><p>Set the validity</p></figcaption></figure>

You also have the option to refresh the expiration time with the next order.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2026-05-18 2355412.png" alt=""><figcaption><p>Refresh expiration time with next order</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2026-05-18 235541224.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

After this, the connections will automatically expire once the validity period ends.

<figure><img src="../../../.gitbook/assets/image (3877).png" alt=""><figcaption></figcaption></figure>
