# Set Connection Validity with Sales

**GoAffPro** provides you with the option to set the customer-affiliate connection validity with sales.&#x20;

To set up the customer-affiliate connection validity with sales, go to the **Extras** section in the **Settings** tab of the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (3878).png" alt=""><figcaption><p>Settings > Extras</p></figcaption></figure>

Here, go to the Customer-Affiliate Connect section.

<figure><img src="../../../.gitbook/assets/image (3879).png" alt=""><figcaption><p>Customer-Affiliate Connect</p></figcaption></figure>

Now, enable the "**Set connection expiration based on number of orders?**" option.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2026-05-19 000831.png" alt=""><figcaption><p>Enable the "Set connection expiration based on number of orders?" option</p></figcaption></figure>

Next, set the connection validity with sales.

{% hint style="info" %}
For example, if you set the validity as five sales, then the customer-affiliate connection will expire after five sales of the connected customer.&#x20;
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2026-05-19 0008352.png" alt=""><figcaption><p>Set the connection validity with sales</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2026-05-19 0008392.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

After this, the connection will automatically expire after the set number of sales is reached.

<figure><img src="../../../.gitbook/assets/image (3880).png" alt=""><figcaption></figcaption></figure>
