# Automatically Apply Discount Codes

**GoAffPro** provides you with the option to automatically apply discount codes for customers.&#x20;

> When a customer visits your store using an affiliate's referral link, the coupon code assigned to the affiliate will automatically get applied at the checkout.

To automatically apply discount codes, go to the **Coupons** tab in the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (3457).png" alt=""><figcaption><p>Coupons tab</p></figcaption></figure>

Here, enable the toggle in the **Automatically apply discount** section.

![Automatically apply discount](<../../.gitbook/assets/Annotation 2020-02-03 171958 (1).png>)

This will enable discount codes to get automatically applied at checkout for customers.
