# Set Customer Segment for Personal Coupons

**GoAffPro** provides you with the option to set the customer segment eligibility for personal coupons.&#x20;

To set the customer segment eligibility for personal coupons, go to the Coupons tab in the GoAffPro admin panel.&#x20;

<figure><img src="../../../.gitbook/assets/image (3853).png" alt=""><figcaption><p>Coupons</p></figcaption></figure>

Here, go to the Personal Coupons section and click on **Change**.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2026-03-18 214723.png" alt=""><figcaption><p>Personal Coupons > Click on Change</p></figcaption></figure>

This will open up the Setup Personal Coupons section.

<figure><img src="../../../.gitbook/assets/image (3856).png" alt=""><figcaption><p>Setup Personal Coupons</p></figcaption></figure>

Here, select the customer segments.

{% hint style="info" %}
You can [create customer segments](https://help.shopify.com/en/manual/customers/customer-segmentation/create-customer-segments) from the Shopify admin panel > Customer > Segments section.
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2026-03-18 214727.png" alt=""><figcaption><p>Select the customer segments</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2026-03-18 214730rw.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>
