# Setup Gifts for Affiliates

**GoAffPro** provides you with the option to set up gifts for your affiliates.

To set up gifts for your affiliates, go to the **Coupons** tab in the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (3729).png" alt=""><figcaption><p>Coupons</p></figcaption></figure>

Here, go to the Gifts section and click on **Create a new template**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-29 125321.png" alt=""><figcaption><p>Gifts > Click on Create a new template</p></figcaption></figure>

This will open up the New Gifts section.

<figure><img src="../../.gitbook/assets/image (3730).png" alt=""><figcaption><p>New Gift</p></figcaption></figure>

Here, set the title.

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-29 125407.png" alt=""><figcaption><p>Set the title</p></figcaption></figure>

### Select Products

Now, click on **Add products**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-29 1304034.png" alt=""><figcaption><p>Add products</p></figcaption></figure>

Next, choose the products.&#x20;

{% hint style="info" %}
You will select the products here to send to your affiliates.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-29 130052.png" alt=""><figcaption><p>Choose the products</p></figcaption></figure>

You also have the option to set the quantity and variation for the products.

<figure><img src="../../.gitbook/assets/image (3734).png" alt=""><figcaption><p>Set the quantity and variation</p></figcaption></figure>

### Gift Settings

After this, go to the Gifts Settings section. Here, you can set options such as:&#x20;

* **Max products:** The maximum number of products that can be claimed.
* **Gift shipping method:** Set the name for the gift shipping method, which will be displayed to affiliates at the checkout.
* **Discount name:** Set the discount name that will apply at the checkout when affiliates place their order.
* **Order tags:** Add a custom order tag to gift orders.&#x20;
* **Billing address:** Add a billing address for gift orders. &#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-29 1255384.png" alt=""><figcaption><p>Setup Gift Settings</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-29 125756.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The gift template will be created. You can now use this template to send gifts to your affiliates.&#x20;

<figure><img src="../../.gitbook/assets/image (3735).png" alt=""><figcaption></figcaption></figure>
