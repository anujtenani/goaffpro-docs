# Set Usage Limit for Automatic Coupons

**GoAffPro** provides you with the option to set the usage limit for automatic coupons. This will limit the total number of times a coupon code can be used.

To set the usage limit for automatic coupons, go to the **Coupons** tab in the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (3455).png" alt=""><figcaption><p>Coupons</p></figcaption></figure>

Here, click on **Change** in the Automatic Coupons section.

![Click on Change](<../../../.gitbook/assets/Annotation 2020-01-27 170705 (1).png>)

This will open up the Setup Automatic Coupons section.

![Setup Automatic Coupons](<../../../.gitbook/assets/Annotation 2020-01-27 171008 (2).png>)

Now, check the "Limit number of times this discount can be used in tota&#x6C;**"** option.

![Check the limit number of times this discount can be used in total option](<../../../.gitbook/assets/Annotation 2020-01-28 023509 (3).png>)

Here, fill in the value for the number of times the discount can be used.

![Fill in the value for the number of times the discount can be used](<../../../.gitbook/assets/Annotation 2020-01-28 025428.png>)

After filling in the value, click on **Submit**.

![](<../../../.gitbook/assets/Annotation 2020-01-28 025645 (2).png>)
