# Set Customer Segment

**GoAffPro** provides you with the option to set the customer segment eligibility for affiliate coupons.

To set the customer segment eligibility for coupons, go to the **Coupons** tab in the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (121).png" alt=""><figcaption><p>Coupons > Automatic Coupons</p></figcaption></figure>

Here, in the Automatic Coupons section, click on **Change**.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-21 165529.png" alt=""><figcaption><p>Click on Change</p></figcaption></figure>

This will open up the Setup Automatic Coupons section.

<figure><img src="../../../.gitbook/assets/image (122).png" alt=""><figcaption><p>Setup Automatic Coupons</p></figcaption></figure>

Now, click on **Advanced**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-21 165538.png" alt=""><figcaption><p>Click on Advanced</p></figcaption></figure>

After this, select the customer segments.

{% hint style="info" %}
You can [create customer segments](https://help.shopify.com/en/manual/customers/customer-segmentation/create-customer-segments) from the Shopify admin panel > Customer > Segments section.&#x20;
{% endhint %}

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-21 165627.png" alt=""><figcaption><p>Select the custom segments</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-07-21 1656573.png" alt=""><figcaption></figcaption></figure>
