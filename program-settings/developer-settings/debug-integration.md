# Debug Integration

**GoAffPro** provides you with the option to debug referral link tracking on your store.

To debug referral link tracking on your store, go to the **Developer** section of the **Settings** tab in the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (2) (1).png" alt=""><figcaption><p>Settings > Developer</p></figcaption></figure>

Here, go to the Debug Integration section.

<figure><img src="../../.gitbook/assets/image (3) (1).png" alt=""><figcaption><p>Debug Integration</p></figcaption></figure>

Now, click on **Start Debugger**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-06-10 213646.png" alt=""><figcaption><p>Click on Start Debugger</p></figcaption></figure>

This will open up the Integration Debugger window.

<figure><img src="../../.gitbook/assets/image (3897).png" alt=""><figcaption><p>Integration Debugger</p></figcaption></figure>

Here, click on **Verify Tracking Script**.

<figure><img src="../../.gitbook/assets/Screenshot 2026-06-10 21370142.png" alt=""><figcaption><p>Click on Verify Tracking Script</p></figcaption></figure>

If the tracking code is not detected, **copy** the embed code and add it to your store theme.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-06-10 21372812.png" alt=""><figcaption><p>Tracking code not detected > Embed code</p></figcaption></figure>

Next, click on **Generate Test Link**.

<figure><img src="../../.gitbook/assets/Screenshot 2026-06-10 213728.png" alt=""><figcaption><p>Click on Generate Test Link</p></figcaption></figure>

After this, open the **test link**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-06-10 213738.png" alt=""><figcaption></figcaption></figure>

A debug popup will open up in the top-right corner of the page.&#x20;

<figure><img src="../../.gitbook/assets/image (3899).png" alt=""><figcaption></figcaption></figure>

Now, place an order and click on **Export** in the popup to download the activity log. If the order is recorded in GoAffPro, it means that the referral link tracking is working properly.

{% hint style="info" %}
If the order was not recorded, please send the exported activity log to our team at admin@goaffpro.com.
{% endhint %}
