# Squarespace/Acuity Scheduling Integration

**GoAffPro** provides you with the option to integrate Squarespace/Acuity scheduling.

To integrate Squarespace/Acuity scheduling, go to the **Integrations** section in the **Settings** tab of the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (3595).png" alt=""><figcaption><p>Settings > Integrations</p></figcaption></figure>

Here, go to the Squarespace/Acuity Scheduling section.

<figure><img src="../../.gitbook/assets/image (651).png" alt=""><figcaption><p>Squarespace/Acuity Scheduling</p></figcaption></figure>

Next, click on **See instructions**.

<figure><img src="../../.gitbook/assets/Screenshot 2023-05-22 220304.png" alt=""><figcaption><p>Click on See instructions.</p></figcaption></figure>

Now, **copy** the tracking code, go to the Home Menu section > Scheduling > Integrations > Analytics & Conversion > Setup for custom conversion tracking > Paste the code in the HTML tracking code field and **Save**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2023-05-22 220340.png" alt=""><figcaption></figcaption></figure>
