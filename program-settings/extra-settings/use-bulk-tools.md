# Use Bulk Tools

**GoAffPro** provides you with the option to use bulk tools to perform bulk actions in the program.&#x20;

To use bulk tools, go to the **Bulk Tools** tab in the GoAffPro admin panel.&#x20;

<figure><img src="../../.gitbook/assets/image (128).png" alt=""><figcaption><p>Bulk Tools</p></figcaption></figure>

Here, click on the **bulk action**.&#x20;

For example: If you want to import customers who shopped at your store before the app was installed, use the "**Import existing customers**" option.&#x20;

You only need to do this once to update the app with customers who shopped before it was installed, as the app tracks customers who come in after installation.

<figure><img src="../../.gitbook/assets/Screenshot 2025-06-10 223348.png" alt=""><figcaption><p>Click on the bulk action</p></figcaption></figure>

This will open up the import customer database section.

<figure><img src="../../.gitbook/assets/image (133).png" alt=""><figcaption><p>Import customer database<br></p></figcaption></figure>

Now, download the customer data from your store.&#x20;

{% hint style="info" %}
For Shopify merchants, you can access it from the Shopify admin panel > Customers > Export option.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2025-06-10 223452g3.png" alt=""><figcaption><p>Download the customer database.</p></figcaption></figure>

The file must include the 'Email' or 'Customer Email' column, which contains all the customer emails.

<figure><img src="../../.gitbook/assets/image (131).png" alt=""><figcaption><p>Customer Email column</p></figcaption></figure>

After this, upload the file.

<figure><img src="../../.gitbook/assets/Screenshot 2025-06-10 223452 (1).png" alt=""><figcaption><p>Upload the file</p></figcaption></figure>

Finally, click on **Submit**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-06-10 2237404.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The customers will be imported into the program.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-06-10 223827.png" alt=""><figcaption></figcaption></figure>
