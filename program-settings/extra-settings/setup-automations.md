# Setup Automations

**GoAffPro** provides you with the option to set up automations for your affiliate program.

To set up automations for your affiliate program, go to the **Automations** tab in the GoAffPro admin panel.&#x20;

<figure><img src="../../.gitbook/assets/image (3720).png" alt=""><figcaption><p>Automations</p></figcaption></figure>

Here, click on **New automation**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-14 120408.png" alt=""><figcaption><p>Click on New automation</p></figcaption></figure>

Now, set the name of the automation.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-14 120437.png" alt=""><figcaption><p>Set the name</p></figcaption></figure>

Next, select the trigger.&#x20;

**You can select triggers such as:**

* New Order
* Runs daily
* Runs on Affiliate Signup

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-14 120507.png" alt=""><figcaption><p>Select the trigger</p></figcaption></figure>

### Add Condition

Here, click on **Add condition**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-14 120556.png" alt=""><figcaption><p>Click on Add condition</p></figcaption></figure>

Now, select the condition.&#x20;

**You can select conditions such as:**&#x20;

* Affiliate's ID
* Affiliate's group ID
* Affiliate Account age
* Affiliate's total net sale value (CV)
* Affiliate's total gross sale value (TOTAL)
* Affiliate's total sale count
* Affiliate's total sale earnings
* Affiliate's total network earnings
* Affiliate's total earnings
* Affiliate's Team Volume
* Product Sold (name of the product)
* Order value

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-14 120628.png" alt=""><figcaption><p>Select the condition</p></figcaption></figure>

Next, set the time period and matching value.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-14 120708.png" alt=""><figcaption><p>Set the time period and matching value</p></figcaption></figure>

### Add Action

Here, click on **Add action**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-14 120750.png" alt=""><figcaption><p>Click on Add action</p></figcaption></figure>

Now, select the action.

**You can select actions such as:**

* Change commission group
* Give discount code
* Give bonus
* Send email
* Apply a tag
* Remove a tag
* Connect customer with the affiliate
* Create affiliate account for customer

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-14 1208124.png" alt=""><figcaption><p>Select the action</p></figcaption></figure>

After this, set up the action.

{% hint style="info" %}
You can configure the action based on the parameter you select. For example, if you choose the action to send an email, you will have the option to select the email template that will be sent to the affiliate once the condition is met.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-14 120942.png" alt=""><figcaption><p>Set up the action</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-04-14 1209424.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The automation will be saved.&#x20;
