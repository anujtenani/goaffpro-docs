# Setup Social Media Campaign Tracking

**GoAffPro** provides you with the option to set up social media campaign tracking for affiliate posts.

To set up social media campaign tracking for affiliate posts, go to the **Social Media** tab in the GoAffPro admin panel.&#x20;

<figure><img src="../../.gitbook/assets/image (265).png" alt=""><figcaption><p>Social Media</p></figcaption></figure>

Here, click on **New campaign**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-01-14 163016.png" alt=""><figcaption><p>Here, click on New campaign</p></figcaption></figure>

Now, set the campaign name.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-01-14 163032.png" alt=""><figcaption><p>Set the campaign name</p></figcaption></figure>

Next, select the affiliate.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-01-14 163110.png" alt=""><figcaption><p>Select the affiliate</p></figcaption></figure>

After this, click on **Submit**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-01-14 163122.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

Now, set the Instagram post URL.&#x20;

{% hint style="info" %}
This will let you track the likes, comments, and reel views for the post.&#x20;
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2025-01-14 163159.png" alt=""><figcaption><p>Set the Instagram post URL</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-01-14 163421.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

You can monitor sales and campaign statistics in the Social Media tab.

<figure><img src="../../.gitbook/assets/Screenshot 2025-01-14 1635113.png" alt=""><figcaption></figcaption></figure>
