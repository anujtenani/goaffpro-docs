# Shopify Buy Button Integration

**GoAffPro** provides you with the option to enable tracking for the Shopify buy button embedded on external sites.

To enable tracking for the Shopify buy button embedded on external sites, go to the **Integrations** section in the **Settings** tab of the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (3906).png" alt=""><figcaption><p>Settings > Integrations</p></figcaption></figure>

Here, go to the Shopify Buy Button section.

<figure><img src="../../.gitbook/assets/image (3907).png" alt=""><figcaption><p>Shopify Buy Button</p></figcaption></figure>

Now, enable the toggle.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-07-01 22163742.png" alt=""><figcaption><p>Enable the toggle</p></figcaption></figure>

After this, copy the tracking code and add it to the pages where you have embedded your Shopify buy button.&#x20;
