# Hyperwallet Integration

**GoAffPro** provides you with the option to integrate Hyperwallet.&#x20;

To integrate Hyperwallet, go to the **Integrations** section of the **Settings** tab in the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (117).png" alt=""><figcaption><p>Settings > Integrations</p></figcaption></figure>

Here, go to the Hyperwallet section.

<figure><img src="../../.gitbook/assets/image (119).png" alt=""><figcaption><p>Hyperwallet </p></figcaption></figure>

Now, click on **Setup**.

<figure><img src="../../.gitbook/assets/image42.png" alt=""><figcaption><p>Click on Setup</p></figcaption></figure>

After this, set the Rest API Username and password. Also, set the program token and ID.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-07-28 1638443.png" alt=""><figcaption><p>Set the Rest API Username &#x26; Password > Set the Program Token &#x26; ID</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-07-28 163937.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

Hyperwallet will now be connected.&#x20;

<figure><img src="../../.gitbook/assets/image (120).png" alt=""><figcaption></figcaption></figure>
