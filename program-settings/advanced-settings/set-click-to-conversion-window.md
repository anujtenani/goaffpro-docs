# Set Click to Conversion Window

**GoAffPro** provides you with the option to set the click to conversion window for your program.

{% hint style="info" %}
The Click-to-Conversion window option allows you to set a timeframe during which a referred customer must make a purchase for the affiliate to receive credit for that sale. It can help in preventing fraud, such as cookie stuffing on the order confirmation page through browser extensions.
{% endhint %}

To set the click to conversion window, go to the **Advanced** section in the **Settings** tab of the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (3806).png" alt=""><figcaption><p>Settings > Advanced</p></figcaption></figure>

Here, go to the Click to conversion window section.

<figure><img src="../../.gitbook/assets/image (3803).png" alt=""><figcaption><p>Click to conversion window</p></figcaption></figure>

Now, click on **Setup**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-09-03 202313.png" alt=""><figcaption><p>Click on Setup</p></figcaption></figure>

This will open up the Update click to conversion window option.&#x20;

<figure><img src="../../.gitbook/assets/image (3804).png" alt=""><figcaption><p>Update click to conversion window</p></figcaption></figure>

Next, you can set the minimum and maximum time.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-09-03 202335.png" alt=""><figcaption><p>Set the minimum and maximum time</p></figcaption></figure>

After this, click on **Submit**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-09-03 2027044.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

Finally, the click to conversion window will be set for the program.&#x20;

<figure><img src="../../.gitbook/assets/image (3805).png" alt=""><figcaption></figcaption></figure>
