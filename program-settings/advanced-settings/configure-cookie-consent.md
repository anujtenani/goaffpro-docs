# Configure Cookie Consent

**GoAffPro** provides you with the option to configure cookie consent for the loading of the tracking code.&#x20;

To configure the cookie consent, go to the **Advanced** section of the **Settings** tab in the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (3806).png" alt=""><figcaption><p>Settings > Advanced</p></figcaption></figure>

Here, go to the Customer cookie consent section.&#x20;

<figure><img src="../../.gitbook/assets/image (279).png" alt=""><figcaption><p>Customer cookie consent</p></figcaption></figure>

Here, you can configure the cookie consent option.&#x20;

{% hint style="info" %}
It is by default set always to load the tracking script.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2024-12-23 1336123.png" alt=""><figcaption><p>Configure the cookie consent option</p></figcaption></figure>

You can set it to load the tracking script only if the customer consents.&#x20;
