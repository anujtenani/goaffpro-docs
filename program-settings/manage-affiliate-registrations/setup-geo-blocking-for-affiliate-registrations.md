# Setup Geo-Blocking for Affiliate Registrations

**GoAffPro** provides you with the option to set up geo-blocking for affiliate registrations.&#x20;

To set up geo-blocking for affiliate registrations, go to the **General** section in the **Settings** tab of the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (3841).png" alt=""><figcaption><p>Settings > General</p></figcaption></figure>

Here, go to the Affiliate Registrations section.

<figure><img src="../../.gitbook/assets/image (3842).png" alt=""><figcaption><p>Affiliate Registrations</p></figcaption></figure>

Now, click on **Setup Geo-Blocking**.

<figure><img src="../../.gitbook/assets/Screenshot 2026-03-04 154206.png" alt=""><figcaption><p>Click on Setup Geo-Blocking</p></figcaption></figure>

This will open up the Signup country blacklist/whitelist window.

<figure><img src="../../.gitbook/assets/image (3843).png" alt=""><figcaption><p>Signup country blacklist/whitelist</p></figcaption></figure>

Next, you can whitelist countries to allow signups only from those locations.

<figure><img src="../../.gitbook/assets/Screenshot 2026-03-04 154226.png" alt=""><figcaption><p>Whitelist countries</p></figcaption></figure>

You also have the option to block signups from specific countries by blacklisting them.​

{% hint style="info" %}
You can set up either a whitelist or a blacklist for countries.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2026-03-04 1542262.png" alt=""><figcaption><p>Blacklist countries</p></figcaption></figure>

Finally, click on **Submit**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2026-03-04 154240.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The geo-block will be set for affiliate registrations in your program.

<figure><img src="../../.gitbook/assets/image (3844).png" alt=""><figcaption></figcaption></figure>
