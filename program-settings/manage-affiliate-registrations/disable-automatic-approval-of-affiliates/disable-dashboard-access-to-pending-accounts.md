# Disable Dashboard Access to Pending Accounts

**GoAffPro** provides you with the option to disable dashboard access to affiliate accounts that are pending approval.

To disable dashboard access to affiliate accounts that are pending approval, go to the **General** section in the **Settings** tab of the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (97).png" alt=""><figcaption><p>Settings > General</p></figcaption></figure>

Here, go to the Affiliate Registrations section.&#x20;

![Affiliate Registrations](<../../../.gitbook/assets/image (1173).png>)

Now, check the Disable dashboard access to pending accounts box.

![Check the Disable dashboard access to pending accounts box](<../../../.gitbook/assets/Annotation 2020-04-07 002453.png>)

After this, affiliate accounts with pending approval will not be able to access their dashboards.

![](<../../../.gitbook/assets/Annotation 2020-04-07 002859.png>)
