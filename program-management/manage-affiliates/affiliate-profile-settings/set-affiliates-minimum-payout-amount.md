# Set Affiliate's Minimum Payout Amount

**GoAffPro** provides you with the option to set an affiliate's minimum payout amount.

To set an affiliate's minimum payout amount, go to the **All Affiliates** section in the **Affiliates** tab of the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (3520).png" alt=""><figcaption><p>Affiliates > All Affiliates</p></figcaption></figure>

Here, click on the **affiliate's name.**

![Click on the affiliate's name](<../../../.gitbook/assets/Annotation 2020-03-03 013129.png>)

This will open up the affiliate's profile.

![Affiliate Profile](<../../../.gitbook/assets/Annotation 2020-03-03 013817 (1).png>)

Now, go to the **Settings** tab.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-04-28 1119384.png" alt=""><figcaption></figcaption></figure>

Next, go to the Payment Settings section and click on Edit.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-04-28 1119387.png" alt=""><figcaption><p>Payment Settings > Click on Edit</p></figcaption></figure>

After this, enter the minimum payout amount.&#x20;

<figure><img src="../../../.gitbook/assets/Screenshot 2025-04-28 111127.png" alt=""><figcaption><p>Enter the minimum payout amount</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-04-28 111207.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The minimum payout amount will be set for the affiliate. They will have to earn this minimum commission amount before becoming eligible for a payout.&#x20;

{% hint style="info" %}
For example, if the minimum payout amount is set to $50, the affiliate must earn a minimum of $50 in commissions before getting paid.
{% endhint %}
