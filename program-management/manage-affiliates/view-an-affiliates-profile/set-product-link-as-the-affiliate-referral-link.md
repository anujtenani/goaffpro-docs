# Set Product Link as Affiliate Referral Link

**GoAffPro** provides you with the option to set a product link as the affiliate's referral link.&#x20;

To set a product link as the affiliate's referral link, go to the **All Affiliates** section in the **Affiliates** tab of the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (143).png" alt=""><figcaption><p>Affiliates > All Affiliates</p></figcaption></figure>

Here, click on the **affiliate's name**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-05-20 221257.png" alt=""><figcaption><p>Click on the affiliate's name</p></figcaption></figure>

This will open up the affiliate's profile.

<figure><img src="../../../.gitbook/assets/image (144).png" alt=""><figcaption><p>Affiliate's profile</p></figcaption></figure>

Now, in the Referral Link section, click on **Customize**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-05-20 221314.png" alt=""><figcaption><p>Click on Customize </p></figcaption></figure>

This will open up the Customize Referral Link window.

<figure><img src="../../../.gitbook/assets/image (145).png" alt=""><figcaption><p>Customize Referral Link</p></figcaption></figure>

After this, paste the product link in the landing page field.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-05-20 221339.png" alt=""><figcaption><p>Paste the product link in the landing page field</p></figcaption></figure>

Finally, click on **Submit**.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-05-20 221411.png" alt=""><figcaption><p>Click on Submit</p></figcaption></figure>

The product link will serve as the affiliate's referral link.

<figure><img src="../../../.gitbook/assets/Screenshot 2025-05-20 2214372.png" alt=""><figcaption></figcaption></figure>
