# View an Affiliate's Profile

**GoAffPro** provides you with the option to view an affiliate's profile. By accessing an affiliate's profile, you can approve or reject their account, set up commissions, coupons, or update their profile information.

To view an affiliate's profile, go to the **All Affiliates** section in the **Affiliates** tab of the GoAffPro admin panel.

<figure><img src="../../../.gitbook/assets/image (3751).png" alt=""><figcaption><p>Affiliates > All Affiliates</p></figcaption></figure>

Here, click on the **affiliate name** to view their profile.

![Click on the affiliate's name](<../../../.gitbook/assets/Annotation%202020-03-03%20013129 (1).png>)

Finally, the affiliate's profile will open up. Here, you have options such as:

* [Assign multiple coupons ](assign-multiple-coupons-to-affiliate.md)
* [Assign personal coupon](assign-personal-coupon-to-affiliate.md)
* [Resend invitation email](resend-invitation-email-to-affiliate.md)

* [Trigger store sync](trigger-store-sync-for-affiliate.md)
* [Send emails to affiliate](send-emails-to-affiliate.md)
* [Verify an affiliate's email](verify-an-affiliates-email-manually.md)
