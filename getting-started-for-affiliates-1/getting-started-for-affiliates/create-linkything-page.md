# Create Linkything Page

In the **Linkything** tab, you can create your link-in-bio page, which generates a shareable landing page that hosts multiple links for use in your social media profile’s bio.

By default, a link in bio page is automatically generated for you. You also have the option to create a new page by clicking on **Create New Page**.

<figure><img src="../../.gitbook/assets/image (62).png" alt=""><figcaption><p>Affiliate Dashboard > Linkything</p></figcaption></figure>

To customize the link in bio page, click on **View Page**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-12 181801.png" alt=""><figcaption><p>Click on View Page</p></figcaption></figure>

Here, you can manage your link in bio page. You can edit the theme and components of the page. You can also view analytics and adjust various page settings.

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-12 181808.png" alt=""><figcaption><p>Manage your link in bio page</p></figcaption></figure>

#### Customize Theme

To customize the theme of the page, click on **Theme**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-12 2017522.png" alt=""><figcaption><p>Customize the theme of the page</p></figcaption></figure>

Here, you can choose the theme for your link in bio page.&#x20;

<figure><img src="../../.gitbook/assets/image (65).png" alt=""><figcaption><p>Choose the theme for the link in bio page</p></figcaption></figure>

#### Edit Page Components

You can also edit the components of the page by clicking on them.

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-12 201746.png" alt=""><figcaption><p>Click on components to edit them</p></figcaption></figure>

Here, you can modify your profile information, like your profile photo, display name, bio, and social links.

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-12 2028232.png" alt=""><figcaption><p>Edit profile information</p></figcaption></figure>

#### View Page Analytics

In the **Analytics** page, you can track the performance of your links and page.&#x20;

<figure><img src="../../.gitbook/assets/image (66).png" alt=""><figcaption><p>Affiliate Dashboard > Linkything > Analytics</p></figcaption></figure>

Here is how the default link in bio page looks like:&#x20;

<figure><img src="../../.gitbook/assets/image (63).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
If the Linkything tab is missing from your dashboard, you can create your link in bio page directly at [linkthing.com](https://linkything.com/).
{% endhint %}
