# Setup Shopping with Popup

**GoAffPro** provides you with the option to setup the shopping with popup.&#x20;

{% hint style="info" %}
The shopping with popup option shows a popup to customers when they visit your store using the affiliate's referral link. You can customize the content displayed in this popup.
{% endhint %}

To set up the shopping with popup, go to the **Toolbars** section in the **Look and Feel** tab of the GoAffPro admin panel.&#x20;

<figure><img src="../../.gitbook/assets/image (86).png" alt=""><figcaption><p>Look and Feel > Toolbars</p></figcaption></figure>

Here, go to the Shopping with Popup section.&#x20;

<figure><img src="../../.gitbook/assets/image (87).png" alt=""><figcaption><p>Shopping with Popup</p></figcaption></figure>

Now, enable the toggle.

{% hint style="info" %}
If you are on Shopify, you can follow this [guide](add-widget-to-store.md) to enable the popup.&#x20;
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2025-09-24 171948.png" alt=""><figcaption><p>Enable the toggle</p></figcaption></figure>

You also have the option to customize the popup by clicking on **Customize**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-09-24 172001.png" alt=""><figcaption><p>Click on Customize</p></figcaption></figure>

Here, you can customize the popup HTML.

<figure><img src="../../.gitbook/assets/Screenshot 2025-09-24 1720204.png" alt=""><figcaption><p>Customize Popup HTML</p></figcaption></figure>

Finally, the popup will be displayed on the store.&#x20;

<figure><img src="../../.gitbook/assets/image (88).png" alt=""><figcaption></figcaption></figure>
