# Setup & Customize Toolbars

The **Toolbars** section can be used to set up and customize toolbars. The options available in this section are:&#x20;

1. [Affiliate Toolbar](setup-affiliate-toolbar/)
2. [Shopping-with Toolbar](setup-shopping-with-toolbar/)
3. [Shopping-with Floating Widget](setup-shopping-with-floating-widget/)
4. [Share A Cart](setup-share-a-cart-button.md)
5. [Shopping Cart Referred By](setup-shopping-cart-referred-by-widget/)
6. [Checkout blocker](setup-checkout-blocker.md)

Go to the **Look and Feel > Toolbars** section in the GoAffPro admin panel.&#x20;
