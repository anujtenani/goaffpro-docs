# Disable Social Media Login

**GoAffPro** provides you with the option to disable the social media logins from the signup page.&#x20;

{% hint style="info" %}
The social media logins can be disabled to prevent affiliates from bypassing the signup fields. &#x20;
{% endhint %}

To disable the social media logins from the signup page, go to the **Signup Page** section in the **Look and Feel** tab of the GoAffPro admin panel.

<figure><img src="../../.gitbook/assets/image (7) (1).png" alt=""><figcaption><p>Look and Feel > Signup page</p></figcaption></figure>

Here, click on **Select & Customize.**&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-07-29 185040.png" alt=""><figcaption><p>Click on Select &#x26; Customize</p></figcaption></figure>

This will open up the signup page section.

<figure><img src="../../.gitbook/assets/image (115).png" alt=""><figcaption><p>Signup page section</p></figcaption></figure>

Now, disable the social media login options.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-07-29 1849363.png" alt=""><figcaption><p>Disable the social media logins</p></figcaption></figure>

Finally, click on **Save**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-07-29 185023.png" alt=""><figcaption><p>Click on Save</p></figcaption></figure>

After this, the social media logins will be removed from the signup page.

<figure><img src="../../.gitbook/assets/image (116).png" alt=""><figcaption></figcaption></figure>
