# Enable Linkything for Affiliates

**GoAffPro** provides you with the option to enable linkything for affiliates.&#x20;

{% hint style="info" %}
Linkything.com is a link-in-bio tool that creates a shareable landing page hosting multiple links, which can be placed as a single URL in a social media profile's bio.
{% endhint %}

To enable linkything for affiliates, go to the **AI Tools** tab of the GoAffPro admin panel.&#x20;

<figure><img src="../../.gitbook/assets/image (58).png" alt=""><figcaption><p>AI Tools</p></figcaption></figure>

Here, go to the linkything.com section.

<figure><img src="../../.gitbook/assets/image (59).png" alt=""><figcaption><p>linkything.com</p></figcaption></figure>

Now, enable the toggle.

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-12 1813742.png" alt=""><figcaption><p>Enable the toggle</p></figcaption></figure>

After this, affiliates will have the ability to create link in bio pages from their dashboard.

{% hint style="info" %}
You can check this [guide](../../getting-started-for-affiliates-1/getting-started-for-affiliates/create-linkything-page.md) on how affiliates can create the link in bio page from their dashboard.&#x20;
{% endhint %}

<figure><img src="../../.gitbook/assets/image (60).png" alt=""><figcaption><p>Affiliates will be able to create link in bio pages</p></figcaption></figure>

You also have the option to set up recommended links for affiliates to add to their link in bio page.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-12 1817173.png" alt=""><figcaption><p>Set up recommended links</p></figcaption></figure>

#### View Affiliate Link in Bio pages

To view the link in bio pages created by your affiliates, click on **See pages created by your affiliates**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-11-12 1817172.png" alt=""><figcaption><p>Click on See pages created by your affiliates</p></figcaption></figure>

Here, you can view the link in bio pages created by your affiliates.&#x20;
