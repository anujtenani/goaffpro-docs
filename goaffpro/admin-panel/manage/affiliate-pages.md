# Affiliate Pages

The **Affiliate Pages** tab allows you to create and host landing pages for your affiliates. Once this option is enabled, affiliates can design unique landing pages for the products they are promoting. They can then share the link to these landing pages with their followers.

### **Enable the Affiliate Pages tab in the admin panel:**

In the **Settings** tab of the GoAffPro admin panel, go to the **Extras** sectio&#x6E;**.**

<figure><img src="../../../.gitbook/assets/image (217).png" alt=""><figcaption><p>Settings > Extras</p></figcaption></figure>

Here in the **Affiliate Page Builder** section, you can enable the toggle.

![Affiliate Page Builder](<../../../.gitbook/assets/Annotation 2020-03-04 221210.png>)

After enabling, the **Affiliate Pages** tab will appear in the admin panel.

<figure><img src="../../../.gitbook/assets/image (218).png" alt=""><figcaption><p>Affiliate Pages tab</p></figcaption></figure>

### Pending Review

In the **Pending Review** section, you can view the pages created by your affiliates. You can choose to approve or reject the pages.

<figure><img src="../../../.gitbook/assets/image (219).png" alt=""><figcaption><p>Pending Review</p></figcaption></figure>

{% hint style="info" %}
Affiliates can create their landing page from the [**Pages**](../../affiliate-dashboard/advanced-tabs/pages.md) tab in their dashboard.
{% endhint %}

### All pages

In the **All pages** section, you can view all the landing pages created by your affiliates. The table displays the title of the landing page, the affiliate who created it, status, and actions.

![Affiliate's Pages > All pages ](<../../../.gitbook/assets/Annotation 2019-11-27 021215.png>)

### Create the page

You have the option to create landing pages for your affiliates. This is useful for providing a default format of the landing page that affiliates can use to build their own.

To create the new page, click on **New Page**. Here, set the page title, select the affiliate, and create the page content.

![New Page](<../../../.gitbook/assets/Annotation 2019-11-27 021738.png>)
