# Bulk Tools

The **Bulk Tools** tab offers you options for bulk operations to manage affiliates in your program. You can bulk import or update sales volume, commissions, affiliates, coupons, etc.

### Enable the Bulk Tools tab in the admin panel:&#x20;

In the **Settings** tab of the GoAffPro admin panel, go to the **Extras** section.

<figure><img src="../../../.gitbook/assets/image (217).png" alt=""><figcaption><p>Settings > Extras</p></figcaption></figure>

Here in the **Bulk Tools**, enable the toggle.&#x20;

<figure><img src="../../../.gitbook/assets/image (186).png" alt=""><figcaption><p>Bulk Tools</p></figcaption></figure>

After enabling, the **Bulk Tools** will appear in the admin panel.

<figure><img src="../../../.gitbook/assets/image (27).png" alt=""><figcaption><p>Bulk Tools tab</p></figcaption></figure>

**In the Bulk Tools section, you can perform the following bulk operations:**

* Update sale volumes or commissions
* Import sales
* Update affiliates
* Import affiliates
* Update coupons
* Import coupons
* Import files
* Add or Update Tags
* Import customer-affiliate connections
* Import existing customers
* Bulk Delete Payouts
* Generate Payment Invoices
* Generate 1099 NEC Forms
* Bulk import balances
