# Social Media

The **Social Media** tab can be used to set up social media campaign tracking. It allows your affiliates to track the performance of their social media campaigns and posts.

### Enable the Social Media tab in the admin panel:

In the **Settings** tab of the GoAffPro admin panel > go to the **Extras** section.

<figure><img src="../../../.gitbook/assets/image (190).png" alt=""><figcaption><p>Settings > Extras</p></figcaption></figure>

Here, in the Social Media Campaigns section, you can enable the **toggle**.

<figure><img src="../../../.gitbook/assets/image (191).png" alt=""><figcaption><p>Social Media Campaigns</p></figcaption></figure>

After enabling, the **Social Media** tab will appear in the admin panel.

<figure><img src="../../../.gitbook/assets/image (192).png" alt=""><figcaption><p>Social Media</p></figcaption></figure>

### Create a new social media campaign

In the Campaigns section, you can create new campaigns, by clicking on **New campaign**.

Here, in the Create new campaign window, you can set the campaign name and select the affiliate. &#x20;

<figure><img src="../../../.gitbook/assets/image (193).png" alt=""><figcaption><p>Create new campaign</p></figcaption></figure>

In the Campaign section, you can set the Instagram URL (to track the links, comments, or reel views) and click on **Submit**.

<figure><img src="../../../.gitbook/assets/image (194).png" alt=""><figcaption><p>Setup the campaign</p></figcaption></figure>

You can track your sales and campaign statistics in the Social Media tab.

<figure><img src="../../../.gitbook/assets/image (195).png" alt=""><figcaption><p>Campaign tracking</p></figcaption></figure>
