# Affiliate Files

The **Affiliate Files** tab allows you to view uploaded files shared with affiliates in your program.

### Enable the Affiliate Files tab in the admin panel:

In the **Look and Feel** tab of the GoAffPro admin panel, go to the **Affiliate Dashboard** sectio&#x6E;**.**

<figure><img src="../../../.gitbook/assets/image (200).png" alt=""><figcaption><p>Look and Feel > Affiliate Dashboard</p></figcaption></figure>

Here, in the Dashboard Tabs section, enable the "**My Files**" option.

![Dashboard Tabs](<../../../.gitbook/assets/image (1331).png>)

After enabling, the **Affiliate Files** tab will appear in the admin panel.

<figure><img src="../../../.gitbook/assets/image (201).png" alt=""><figcaption><p>Affiliate Files</p></figcaption></figure>

### View Affiliate Files

In the **Affiliate Files** tab, you can view and download the files uploaded or shared with affiliates.&#x20;

You can search for files by affiliate name. The table displays the file name, the affiliate with whom it is shared, the date it was uploaded, and relevant actions such as deleting the file.

<figure><img src="../../../.gitbook/assets/image (203).png" alt=""><figcaption><p>View affiliate files</p></figcaption></figure>

You also have the option to bulk upload files for your affiliates by clicking on **Bulk upload files**.&#x20;

<figure><img src="../../../.gitbook/assets/image (205).png" alt=""><figcaption><p>Bulk upload files</p></figcaption></figure>
