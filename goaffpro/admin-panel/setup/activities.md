# Activities

The **Activities** tab is used to create tasks for affiliates and reward them upon task completion.

### Enable the Activities tab in the admin panel:

In the **Settings** tab of the GoAffPro admin panel, go to the **Extras** section.

<figure><img src="../../../.gitbook/assets/image (244).png" alt=""><figcaption><p>Settings > Extras</p></figcaption></figure>

Here, in the Affiliate tasks and activities section, you can enable the **toggle**.

<figure><img src="../../../.gitbook/assets/image (3357).png" alt=""><figcaption><p>Affiliate tasks and activities</p></figcaption></figure>

After enabling, the **Activities** tab will appear in the admin panel.

<figure><img src="../../../.gitbook/assets/image (245).png" alt=""><figcaption><p>Activities</p></figcaption></figure>

### Create activity sheet

In the **Activities** tab, you can create task sheets for your affiliates. To create a new activity sheet, click on **New activity sheet**.

<figure><img src="../../../.gitbook/assets/image (3358).png" alt=""><figcaption><p>Affiliate Tasks</p></figcaption></figure>

In the Sheet section, you can add activities for affiliates to complete.

<figure><img src="../../../.gitbook/assets/image (3359).png" alt=""><figcaption><p>Activity Sheet</p></figcaption></figure>

In the Add activity window, you can set the category, title, tag, and description for the activity. You can also set up the reward for the activity by setting the payout type, amount, maximum allowed, and units.&#x20;

<figure><img src="../../../.gitbook/assets/image (3360).png" alt=""><figcaption><p>Add Activity</p></figcaption></figure>

### Add activity page to the affiliate dashboard

After this, you can add the activity page to the affiliate dashboard by creating a new dashboard page. You can go to the Look and Feel > Affiliate Dashboard > Dashboard Tabs > New Custom Page section.&#x20;

To add the activity sheet to the affiliate dashboard, click on **Add section** and select the Affiliate Activity Sheet option.

<figure><img src="../../../.gitbook/assets/Screenshot 2023-12-11 161920.png" alt=""><figcaption><p>Look and Feel > Affiliate Dashboard > Dashboard Tabs > New Custom Page > Add section > Affiliate Activity Sheet</p></figcaption></figure>

The activity page will be displayed on the affiliate dashboard.

<figure><img src="../../../.gitbook/assets/image (3372).png" alt=""><figcaption><p>The activity page will be displayed on the affiliate dashboard</p></figcaption></figure>
