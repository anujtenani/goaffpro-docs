# Coupons

Coupons are a great way to provide discounts, especially for first-time customers looking to try your brand. They incentivize initial purchases, attract new customers, and encourage existing ones to buy more, promoting higher-value products.

> Coupons are especially beneficial when shared by affiliates on social media platforms like Instagram, TikTok, or YouTube.

In the **Coupons** tab, you will see the **Coupon Based Commissions** section. Here, you can view the coupons assigned to affiliates. When a customer makes a purchase using an affiliate's coupon, the affiliate will earn a commission on that sale. The table displays the affiliate name, coupon code, and the type of coupon.&#x20;

<figure><img src="../../../.gitbook/assets/image (3673).png" alt=""><figcaption><p>Coupons tab</p></figcaption></figure>

### Assign Coupon to Affiliates

To assign a coupon to an affiliate, click on **Assign coupon to affiliate.** Here, you can select the partner/affiliate and set the coupon code. Click on **Submit** to assign the coupon to the affiliate.

![Assign Coupon to Affiliates](<../../../.gitbook/assets/image (697).png>)

### Automatic Coupons

**Automatic Coupons** can be utilized to generate coupons for affiliates automatically.

![Automatic Coupons](<../../../.gitbook/assets/image (1783).png>)

{% content-ref url="../../../affiliate-coupons/setup-referral-coupons/automatically-generate-coupons/" %}
[automatically-generate-coupons](../../../affiliate-coupons/setup-referral-coupons/automatically-generate-coupons/)
{% endcontent-ref %}

You can modify the discount value of the coupons by clicking on **Change**. In this section, you can adjust the discount amount, type, and more. Additionally, you have the option to set usage limits, such as one use per customer, the total number of times the coupon can be redeemed, or restricting coupon use to new customers only.

For more advanced settings, open the **Advanced** section. Here, you can set options such as customer segments, minimum cart quantity, minimum order value, restricting discounts to specific products or collections, setting combination rules, and enabling coupons for subscriptions.

<figure><img src="../../../.gitbook/assets/image (113).png" alt=""><figcaption><p>Automatic Coupons</p></figcaption></figure>

#### Automatic Coupon Style

With the **Coupon Style** option, you can choose the style in which the coupons are generated. In the Automatic Coupons section, click on **Change Style**.

![Change Style](<../../../.gitbook/assets/Screenshot 2019-08-04 at 4.29.52 AM.png>)

In the **Automatic Coupons Style** window, you will be provided with the option to change the style of the coupon code. There are four types of coupon code styles you can choose from:

* **Affiliate's Name**: The coupon code gets generated using the affiliate's name.
* **Affiliate's Instagram Handle**: The coupon code gets generated according to the affiliate's Instagram handle.
* **Random Letters**: The coupon gets generated with random letters.
* **Random Dictionary Words**: The coupon code will be made up of random words.
* **Custom style**: You can create your own custom style for coupon codes.

![Coupon Code Styles](<../../../.gitbook/assets/Untitled (3).png>)

### Dynamic Coupons

The **Dynamic Coupons** option automatically generates a new coupon code for every affiliate. The coupons are applied automatically using the affiliate's link and are refreshed after a certain period.&#x20;

### Automatically apply discount

The **Automatically apply discount** option can be used to apply the discount codes automatically at checkout when a customer visits the store using an affiliate's referral link.

<figure><img src="../../../.gitbook/assets/image (2170).png" alt=""><figcaption><p>Automatically apply discount</p></figcaption></figure>

There is also an option to set a **default coupon code**, which will automatically get applied to the referral links of affiliates without an assigned coupon.

### Detect Coupon Leaks

The **Detect Coupon Leaks** feature notifies you when an affiliate coupon code gets leaked on coupon sites. To view the leaked coupon codes, click on **View leaked coupons**.

<figure><img src="../../../.gitbook/assets/image (2281).png" alt=""><figcaption><p>Detect Coupon Leaks</p></figcaption></figure>

Here, you can view the leaked coupon codes. After taking the appropriate action, you can click on **Mark as actioned** to remove the code's notification.&#x20;

<figure><img src="../../../.gitbook/assets/image (2520).png" alt=""><figcaption><p>Leaked Coupon Codes</p></figcaption></figure>

{% endcontent-ref %}

### Personal Coupons

The **Personal Coupons** option can be used to generate personal coupons for affiliates automatically. Any purchase made using these personal coupons will not generate any commission for the affiliate.

<figure><img src="../../../.gitbook/assets/image (1877).png" alt=""><figcaption><p>Personal Coupons</p></figcaption></figure>

To change the discount type or value of the personal coupons, you can click on **Change**. Here, you can set the discount type and discount value. You can also set usage limits for the personal codes.&#x20;

<figure><img src="../../../.gitbook/assets/image (19).png" alt=""><figcaption><p>Setup Personal Coupons</p></figcaption></figure>

### Gifts

The **Gifts** feature allows you to create gift templates that can be used to send gifts to your affiliates.

<figure><img src="../../../.gitbook/assets/image (153).png" alt=""><figcaption><p>Gifts</p></figcaption></figure>

To configure a gift template, click on **Create a new template**. Here, you can set the title for the gift template and select the products you want to send to affiliates.&#x20;

Additionally, you can configure options for the template, including the maximum number of products that can be claimed, the shipping method, a discount name, an order tag, and the billing address.

<figure><img src="../../../.gitbook/assets/image (154).png" alt=""><figcaption><p>Gift template configuration</p></figcaption></figure>
