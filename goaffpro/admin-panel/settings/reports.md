# Reports

**Reports** section can be used to view and download reports on affiliates, sales, payouts, and more.

<figure><img src="../../../.gitbook/assets/image (213).png" alt=""><figcaption><p>Reports</p></figcaption></figure>

The types of data reports, that you can download are:

* Affiliates&#x20;
* Orders/Sales
* Rewards/Bonuses
* Products
* Payouts&#x20;
* Traffic
* Points
* Connected Customer
* Party/Event report
* W9 report
* Transactions log
* Leads

You can further filter data by affiliate, group, tag, order count, signup dates, etc., and download data reports for them.

{% hint style="info" %}
You can open the CSV file in spreadsheet programs such as Microsoft Excel or Google Sheets.
{% endhint %}
