# Party Links

The **Party Links** tab allows affiliates to get a customer or host to have a party, where the host shares their party link with their friends, family, or guests.

When someone orders using the party link, the host gets a reward, and the affiliate (who created the host and assigned them the party link) earns their commission.

![Party Links tab](<../../../.gitbook/assets/image (2879).png>)

### Create Party

Affiliates can click on **New party**. They will enter the host's name, email address, set the party code, and choose the start date.

<figure><img src="../../../.gitbook/assets/image (162).png" alt=""><figcaption><p>New Party</p></figcaption></figure>

{% hint style="info" %}
The **Party Links** tab will only be displayed to affiliates if the merchant enables and sets up the [party plans](../../../program-settings/extra-settings/setup-party-plans/) feature.&#x20;
{% endhint %}
