# Create New Signup Fields

**GoAffPro** provides you with the option to create new signup fields for the signup page of the affiliate portal.

To create new signup fields for the signup page, go to the **Signup page** section in the **Look and Feel** tab of the GoAffPro admin panel.&#x20;

<figure><img src="../../../.gitbook/assets/image (7).png" alt=""><figcaption><p>Look and Feel > Signup page</p></figcaption></figure>

Here, click on **Select & Customize** for the layout you are using.

<figure><img src="../../../.gitbook/assets/Screenshot 2024-11-11 131229 (1).png" alt=""><figcaption><p>Click on Select and Customize</p></figcaption></figure>

Now, click on **Add your own field**.

<figure><img src="../../../.gitbook/assets/Screenshot 2024-11-11 130319.png" alt=""><figcaption><p>Click on Add your own field</p></figcaption></figure>

This will open up the Signup Field section.

<figure><img src="../../../.gitbook/assets/image (9).png" alt=""><figcaption><p>Signup Field</p></figcaption></figure>

Here, set the field title.

<figure><img src="../../../.gitbook/assets/Screenshot 2024-11-11 130334.png" alt=""><figcaption><p>Set the field title</p></figcaption></figure>

You also have the option to set the signup field as required, change the input type, and adjust the width of the field.

<figure><img src="../../../.gitbook/assets/Screenshot 2024-11-11 130353.png" alt=""><figcaption><p>Additional options</p></figcaption></figure>

After this, click on **Add**.

<figure><img src="../../../.gitbook/assets/Screenshot 2024-11-11 130416.png" alt=""><figcaption><p>Click on Add</p></figcaption></figure>

Finally, click on **Save**.

<figure><img src="../../../.gitbook/assets/Screenshot 2024-11-11 130446f.png" alt=""><figcaption><p>Click on Save</p></figcaption></figure>

The newly created signup field will be displayed on the affiliate portal's signup page.

<figure><img src="../../../.gitbook/assets/image (10).png" alt=""><figcaption></figcaption></figure>
