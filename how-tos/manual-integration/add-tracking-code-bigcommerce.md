# Manual BigCommerce Integration

#### Add the conversion tracking code in your BigCommerce store:

Open the BigCommerce Admin page.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-10-14 165639.png" alt=""><figcaption><p>Open the BigCommerce Admin page</p></figcaption></figure>

Now, go to the Settings section.

<figure><img src="../../.gitbook/assets/Screenshot 2025-10-14 165731.png" alt=""><figcaption><p>Go to the Settings section</p></figcaption></figure>

Next, in the Advanced section, select **Data solutions**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-10-14 142934.png" alt=""><figcaption><p>Setting > Advanced > Data solutions</p></figcaption></figure>

In the Affiliate Conversion Tracking option, click on **Connect**.&#x20;

<figure><img src="../../.gitbook/assets/Screenshot 2025-10-14 165858.png" alt=""><figcaption><p>Affiliate Conversion Tracking > Click on Connect</p></figcaption></figure>

After this, paste the affiliate conversion tracking code.&#x20;

{% hint style="info" %}
You will get this code at the [app.goaffpro.com/wizard](https://app.goaffpro.com/wizard) page.
{% endhint %}

<figure><img src="../../.gitbook/assets/Screenshot 2025-10-14 160423.png" alt=""><figcaption><p>Paste the affiliate conversion tracking code</p></figcaption></figure>

Finally, click on **Connect**.

<figure><img src="../../.gitbook/assets/Screenshot 2025-10-14 1604234.png" alt=""><figcaption></figcaption></figure>
